The Aside Element

The aside element is used to mark additional info that can enhance another element, but
isn't required in order to understand the main content. Aside can be used alongside other
elements such as article or section. Here are a few common uses for aside:

  • Bibliographys
  • Endnotes
  • Comments
  • Pull Quotes
  • Editorial Sidebars
  • Additional Information
