Main & Footer

Two more structural elements are "main & footer". These elements along with "nav" and "header"
help describe where an element is located based on conventional web development standards.


The element"main" is used to encapsulate dominant content within a webpage. This tag is seperate
from the "footer" and the "nav" of a webpage since these elements don't contain the
principal content.

By using "main" instead of "div" screen readers and website crawlers are able to easily
identify what content is inside of the tags.

How does "main" look when incorporated into our code?

Look at the source code below:

If we look into the source code above, we can see "main" contains an "article"
and "header" tag with child elements that hold the most important info related to the page.

The content at the bottom of the subject information is known as the "footer" indicated by the
footer element. The footer contains info such as:

  • Contact information
  • Copyright information
  • Terms of use
  • Site map
  • Reference to top of page links
