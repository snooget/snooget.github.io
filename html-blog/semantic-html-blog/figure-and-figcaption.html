Figure & Figcaption

With aside we learned that we can put additional information next to a main peice of content,
but what if we wanted to add info next to an image?

When adding captions or other media next to images, you'll find figure and figcaption
very useful.

figure is an element used to encapsulate media such as images, illustrations, diagrams,
and code snippets.

In the code above, we created a figure element so that we can encapsulate our image tag.

It is also possible to add a caption to the image by using "figcaption".

Figcaption is an element used to describe the media inside the figure tag. Usuallyfigcaption
will go inside figure This is quite different than using a p element.

The difference helps when changing the location of figure, as a normal paragraph tag would get displaced.
