Table Footers (tfoot) Notes

The (tfoot) element can be used to section off the bottom part of a
long table.


Here is an example (Check the source code):


Quarter Revenue Costs
Q1 $10M $7.5M
Q2 $12M $5M
Total $22M $12.5M

In the example above, the footer contains the totals of the data in the table. Footers are often used to contain
sums, differences, and other data results.
