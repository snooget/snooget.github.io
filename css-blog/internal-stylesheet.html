Internal Stylesheets

As shown in the last lesson, using inline styles is not effective nor efficient. For example, lets say you wanted to style h1 elements for your website. If you were using last lessons inline method you would have to add the CSS code to the beginning tag of each element avery time a h1 tag was inserted in the code. By using internal stylesheets, you can edit multiple or single html elements to your liking.
