CSS Setup and Syntax Review

Learning to implement CSS into my HTML files will aid in the creation of some stunning websites with great user interface and design.

Here is what we have reviewed so far:

  • The basic anatomy of CSS syntax written for inline styles and stylesheets.
  • Some commonly used CSS terms such as ruleset, selector, and declaration.
  • CSS inline styles can be written inside the opening tag using the style attribute.
  • Inline styles can be used to style HTML, but it is not the best practice.
  • An internal stylesheet is written using the style element inside the head element of an HTML file.
  • An internal stylesheet can be used to style HTML but is also not the best practice.
