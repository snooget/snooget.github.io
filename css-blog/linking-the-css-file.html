Linking the CSS Files

Now that we know about external CSS files and have seperated the CSS from the html, it is time to link the pages! CSS won't load in the html file if the two are not linked. To do so, we will create a link element in the head of the html file. We then use the rel and href attributes to complete the link.

  • href: An href is used to connect the address or path of the CSS file.
  • rel: describes the relationship between the html files and the CSS file. Due to the fact that you are linking a stylesheet the value should be set to stylesheet.
