Universal Selectors

Previously, you learned how a type selector selects all elements of a given type. Well, the universal selector selects elements of any type.

Targeting all the elements on the page has a few specific use cases, such as resetting default browser styling, or selecting all children of a parent element. Don't worry if you cant understand the use cases at the moment. They will be covered in later lessons.
