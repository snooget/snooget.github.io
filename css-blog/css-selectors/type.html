Type (CSS)

Remember that declarations are an important part of CSS since they apply a style to a selected element. However, how do we decide which elements will be styled?

In the cases where we would like to specify which elements are to be styled we will use a selector. One selector we may already be familiar with is the type selector. Much like the name suggests, the type selector mathces the type of the element in the HTML document.
