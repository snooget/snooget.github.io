Environment Review

We just learned how to use the bash profile to configure our environment. What can we generalize so far?

  • The environment refers to the prefrences and settings of the current user.
  • The nano editor is a command line text editor used to configure the environment.
  • ~/.bash_profile is where environment settings are stored. This file can be edited
    with nano.
  • Environment variables are variables that can be used across commands and programs to info
    about the environment.
    • $ export VARIABLE="Value" sets and exports an environment variable.
    • USER is the name of the current user.
    • PS1 is the command prompt.
    • HOME is the home directory. It is usually not customized.
    • PATH returns a colon : seperated list of file points.
    • env returns a list of environmental variables. Output can be redirected using grep
      to select the variable we would like to view.
