Configuring the Command Line Intro

What's the point of the Command Line?

The comamnd line is a super useful tool for developing projects. The command line can be used to
manage all project files; you can move files, delete files, reorganize files, search & replace terms,
and so much more!!


You can also control the execution of programs you write and run from dev software.


As I become better at using some of the many programming languages and command line,
I will be able to have many things running at once. For example, I can execute a Python
project and keep track of output as the program runs, I can edit files directly inside
the command line, and I can run commands from development software like Git, to keep things
organized. As I become a better developer I will want to edit the command line environmenr to
my liking and convineince.


What will be taught in these next lessons?

In these lessons I will show how to;

  • Use nano, a keyboard-only text editor that exists only in command line.
  • Use the Bash profile.
  • How to store shorthands, (called alias) for commonly used commands.
  • How to change the way your command prompt looks.
  • How to establish and access global command-line variables.
  • What PATH variable is and how to check which directories executable commands come from.
