Bash Profiles & Aliases

A bash profile is used to store environment settings for your terminal. It is accessible by the name
~/.bash_profile.

When a session starts, it loads the contents of the bash profile before executing commands.

  • The ~ represents the users home directory.
  • The . indicates a hidden file.
  • The name ~/.bash_profileis important. Since this is how the command li
    -ne reconizes the bash profile.

To open and edit the bash profile we will type:

$ ~/.bash_profile

Aliases Notes

As mentioned earlier, settings and commands can be added everytime a new terminal session is started
One setting that can be created to save commands and setting options is the: alias.

$ alias pd="pwd"

The alias command allows you to create shortcuts for commonly used commands. In the command above,
$ alias pd="pwd" creates the alias for the pwd command. It is then
saved in bash profiles.

The pd alias will be avaliable next time we open a new temrinal session, and the
output will be the same as the "pwd" command.

Aliases II

We can add as many aliases as we want into a bash profile. Here are a few examples.

$ alias hy="history"

In the code above, hy is set as an alias for the history command. Typing "hy" in the command line outputs
a history of commands entered in the current session.

$ alias ll="ls -la"

In the code above ll is set as an alias for ls -la.
