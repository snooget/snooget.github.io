Intro to Environment

Each time we launch the terminal application, a new session is created.

The session will load up settings and preferences that make the command-line environment.

We are able to configure the command line to support the command and programs we create.

Typing $ nano hello.txt activates the nano text editor. I can also use crtl + o to savea file
and use ctrl + x to exit nano.

Nano Continued

Here's how nano works:

$ nano hello.txt

Nano is a command line text editor. It works the same way as the notes app in most computers and
phones. The only difference is that it is accessable from the command line.
Let's walkthrough the previous above excercise I did for nano;

  • The command $ nano hello.txt opens a new text file named "hello.txt" in the
    nano text editor.
  • "Hello, I am nano." is a text string entered into at the line indicated by the
    cursor.
  • The menu of keyboard commands at the bottom of the window allow us to save changes to "hello.txt"
    and exit nano. The ^ stands for crtl key.
  • ^ + O will save a file. O stands for output. Again, not case sensitive.
  • ^ + X will exit the nano program. X stands for exit.
  • ^ + G will open the help menu.
