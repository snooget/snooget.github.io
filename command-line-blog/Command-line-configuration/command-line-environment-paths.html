Path Environment Variable

Path is a environment variable that stores a list of directories seperated by a colon.

Typing $ echo $PATH will reveal a long list of directories seperated
by a colon.

Each directory will contain scripts for the command line to execute. The PATHvariable
will simply list which directories contain scripts.

Many of the commands I have learned are stored in the /bin directory.

$ /bin/pwd is the script executed when typing the pwd command.

$ /bin/ls is the script executed when typing the ls command.
