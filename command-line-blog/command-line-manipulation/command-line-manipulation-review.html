Command Line Manipulation Review Notes

So far I have learned:

So far I have learned how to use the command line to view and manipulate the filesystem.

Here is what can be generalized so far:

  • Options modify the behavior of commands

  • ($ ls -a) Lists all contents of a directory including hidden files in that directory.
  • ($ ls -l) Lists all conent in long format.
  • ($ ls -t) Orders files and directories by the date they were last modified.
  • Multiple options can be used together, like ($ ls -alt).

  • From the command line you can also copy, move and remove files and directories.
  • ($ cp) Copies files.
  • ($ mv) Moves and renames files.
  • ($ rm) Removes files.
  • ($ rm -r) Removes directories

  • Wildcards are useful for selecting groups of files and directories.

Quick Questions:


  1. How would you delete a directory called music/?

    2. Answer: $ rm -r music/ (Remember that -r deletes directories and child directories.


  2. What command shows all files in the current directory in long format
    including hidden files?

    3. Answer: $ ls -al (-a option shows contents of a directory including ones that start with ..
    which are typically hidden. However, the -l option will also show content in long format.


  3. Which option allows you to list files by when they were last modified?

    4. Answer: $ ls -t (Shows files based on date last modified.


  4. Which command could you use to copy the contents of kendrick.txt
    to a new file called rappers.txt?

    5. Answer: $ cp kendrick.txt rappers.txt


  5. How would you delete all files in the directory media/ (but not the directory itself)?

    6. Answer: $ rm media/* (The rm command on its own cannot remove a directory, and the second
    argument does not specify the files inside media/)


  6. What is the purpose of an option?

    7. Answer: To modify the behavior of a command


  7. What command moves all files in the current directory to the directory school/?

    8. Answer: $ mv * school/ (mv takes the first argument * to mean all files in the current
    directory, and moves them to the destination school/ specified by the second argument)
