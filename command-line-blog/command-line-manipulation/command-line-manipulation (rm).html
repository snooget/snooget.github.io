Command Line Manipulation (rm) Notes

What does the (rm) command do?


The (rm) command removes files from the file stsyem.
Be careful when using (rm) as it will permanently delete files.

Here's an example of the (rm) command in action:

$ rm unwanted_file.txt


The (rm) command also has a (-r) option that removes
directories and child directories associated with it.

Here's an example:

$ -r unwanted_directory
