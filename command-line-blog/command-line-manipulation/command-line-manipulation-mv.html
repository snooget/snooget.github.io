Command Line Manipulation (mv)

Mv

The (mv) command moves files. It's similar to (cp) in its usage. However, (mv) does not copy files.

To move a file into a directory use (mv) with the source file as the first argument and the
destination directory.


Here is a coded example:

$ mv my_file.txt my_directory/

To move multiple files into a directory it will be written out the same way
with a list of source files as the first arguments and the destination directory as the last argument.


Here's an example of multiple files being moved:

$ mv my_file1.txt my_file2.txt my_directory/


We can also rename a file using the (mv) command. To rename a file use (mv) with the old file
as the first argument and the new file as the second argument.


Here's an example of renaming a file with the (mv) command

$ mv file_origin.txt file_renamed.txt
