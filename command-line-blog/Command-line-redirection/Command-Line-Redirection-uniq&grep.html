uniq & grep | commands

$ uniq deserts.txt

uniq stands for "unique". It filters out adjacent, duplicate lines in a file. Here uniq deserts.txt filters out
duplicates of "sahara desert", because its duplicate directly follows the previous instance. The "kalahari desert"
duplicates are not adjacent and thus remain.

$ sort deserts.txt | uniq

A more effective way to use uniq is to call sort to alphabetize a file, and "pipe" the standard output to uniq.
By "piping" $ sort deserts.txt to uniq all duplicate lines are alphabetized and filtered out.

$ sort deserts.txt | uniq > uniq-deserts.txt

Here we simply send filtered contents to uniq-deserts.txt which you can view with the cat command.


grep command & grep -i option

$ grep America continents.txt

grep stands for "global regular expression print" Its purpose is to search files for lines that match a
pattern and then return the results.
grep is case-sensitive. In the bolded commands above we use grep to search for anything that matched "America"
in continents.txt.

$ grep -i America continents.txt

The -i option for grep allows the command to be case insensitive. Here, grep searched for capital of lowercase
strings that match "America" in the continents.txt file.


grep II

$ grep -R Arctic
/home/ccuser/workspace/geography

/home/ccuser/workspace/geography/deserts.txt: Arctic Desert
/home/ccuser/workspace/geography/oceans.txt: Arctic Ocean

The grep -r option searches for all files in a directory and outputs filenames and lines containing all
matches results. The -r stands for "recursive"

In the commands above grep -r searched the /home/ccuser/workspace/geography for the string "Arctic" and
outputted filenames and lines with matching results.


$ grep -Rl Arctic
/home/ccuser/workspace/geography

/home/ccuser/workspace/geography/deserts.txt
/home/ccuser/workspace/geography/oceans.txt

grep -Rl searches all files in a directory and outputs only filenames with matched results (no lines). Above,
grep -Rl searched the /home/ccuser/workspace/geography directory for the string "Arctic" and outputs filenames wi
-th matched results.
