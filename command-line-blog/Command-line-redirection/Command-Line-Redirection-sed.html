The (sed) Command

$ sed 's/snow/rain/' forests.txt

sed stands for "stream editor". It accepts standard input and modifies it based on the expression, before
displaying it as output data. It is similar to find and replace.

Looking at the expression '/snow/rain/':

  • s stands for "substituition." It is always used when using (sed) for substituition.
  • snow: the search string or the text to find.
  • rain: the replacement string, or text to add in place.

In the code above (sed) searches forests.txt for the word "snow" and replaces it with rain
globally. This means all instances of snow on a line will be turned to rain.

So far we have only used (sed) to rewrite the command line output and the actual file wont look different.
To make actual changes to a file we must use the -i option with (sed). Here's and exmaple:

$ sed -i 's/snow/rain/g' forests.txt

The command above rewrites forests.txt and replaces all instances of snow with rain.
