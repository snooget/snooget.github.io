git merge

We will use git merge to integrate out origin/main branch into our local main branch.

Below is the command:

$ git merge origin/main

Git Workflow

Now that origin/master have been merged into the local master branch we are ready to contribute to some work on
our own. The workflow for Git typically follows this order:

  1. Fetch and merge changes from the remote.
  2. Create a branch to work on a new project feature.
  3. Develop the feature on your branch and commit your work.
  4. Fetch and merge from the remote agin (in case new commits where made while you where working.
  5. Push your branch up to the remote for view.

Steps 1 and 4 are safegaurds against merge conflicts. Step 5 involves a git push command which will be learned next.
