Styling with CSS


Tables, by default are very bland. They don't have borders, the font color is black, and the typeface is the
same used for other html elements.

CSS or Cascading Style Sheets, is a language that web developers use to style the html content on a web page.
CSS can be used to style tables and the various aspects mentioned above.

table, th, td {
border: 1px solid black;
font-family: Arial, sans-serif;
text-align: center;
}

The code above demonstrates some of the various table aspects that you can style using CSS properties.
