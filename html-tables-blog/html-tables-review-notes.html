html Tables Review

Tables are great ways to store and convey data to users, and yourself. This is what I know about tables
with html so far:

  • The .table. element creates a table.

  • The tr element adds rows to a table.

  • To add data to a row, you can use the td element.

  • Use th to add table headings. This brings clarity to your data.
