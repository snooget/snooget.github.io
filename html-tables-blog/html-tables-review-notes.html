html Tables Review

Tables are great ways to store and convey data to users, and yourself. This is what I know about tables
with html so far:

  • The .table. element creates a table.

  • The tr element adds rows to a table.

  • To add data to a row, you can use the td element.

  • Use th to add table headings. This brings clarity to your data.

  • Table data can span columns using the colspan attribute.

  • Table data can span rows using the rowspan attribute.

  • Tables can be split into three main sections: a head, a body, and a footer.

  • A tables head is created with a thead element.

  • A tables body is created with the tbody element.

  • A tables footer is created with the tfoot element.

  • All CSS properties you learned about in this lesson can be applied to tables and their data.
